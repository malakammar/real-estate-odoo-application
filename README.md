# Real Estate Management System (Odoo)

A custom Odoo module developed for managing real estate operations, including property listings, offers, and customer interactions.

## Features

- Property Management
- Property Types & Tags
- Offer Management
- Customer Tracking
- Security Groups & Access Rights
- Form, Tree, and Search Views
- Business Logic using Odoo ORM
- PostgreSQL Database Integration

## Technologies

- Odoo
- Python
- PostgreSQL
- XML
- Odoo ORM
- Kanban Views
- Reports
- QWeb
- OWL

- Models

- Business Logic
